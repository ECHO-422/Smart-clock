#### 3、通过屏幕设置闹钟时间功能

点击屏幕Set Alarm按钮，将会出现以下界面：

![img](./pic/4.jpg)

选择闹钟的时间，进行ADD操作，ADD之后，点击OK按键即为设置成功！

通过串口通信，将设置闹钟时间的情况显示到XCOM软件中。验证设置闹钟时间成功。

![img](./pic/5.jpg)
